# Программа для поиска слова в текстовых файлах

## Описание

Эта программа на языке C предназначена для рекурсивного поиска заданного слова в текстовых файлах, расположенных в директории ~/files и ее вложенных подкаталогах, включая скрытые файлы и директории (начинающиеся с точки).

Программа использует системные вызовы и функции: `opendir()`, `readdir()`, `closedir()`, `fopen()`, `fgets()`, и `strstr()` для поиска в строках файла.

## Особенности реализации

- Рекурсивный обход директорий: Программа использует рекурсию для обхода всех директорий и подкаталогов, включая скрытые.
- Поиск по слову: Программа находит все строки, содержащие заданное слово.
- Обработка ошибок: Программа корректно обрабатывает ошибки, такие как отсутствие файлов, ошибки доступа и другие.

## Используемые технологии

- C: Язык программирования для написания программы.
- pthread: Для многозадачности и синхронизации потоков.
- CMake: Для автоматической сборки проекта.

## Структура проекта

- src/: Папка с исходными файлами проекта.
  - wordsearch.c: Основной исходный код программы.
- CMakeLists.txt: Файл для сборки проекта с помощью CMake.
- README.md: Документация по проекту.
- .gitignore: Файл для игнорирования временных и скомпилированных файлов в Git.
